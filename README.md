# Welcome to Your Digital Closet! 

Hi there! Welcome to your very own **Digital Studio** — a cozy little corner of the internet where your wardrobe lives, breathes, and gets a whole lot more organized. Whether you're a fashion lover, a chronic over-packer, or someone who just wants to stop staring blankly at a full closet thinking "I have nothing to wear," this app is for you!

---

## What is this?

**Digital Studio** is a personal digital closet management app that lets you:

- **Upload and catalog** your clothing items with photos and details
- **Browse your closet** in a beautiful, organized grid
- **Generate outfit ideas** automatically from what you already own
- **Save your favorite outfits** so you never forget that perfect combination again

No more forgetting you own that cute top buried in the back of your wardrobe. It's all here, neatly organized and ready to inspire!

---

## Why did I build this?

I built **Digital Studio** because I genuinely struggled with keeping track of my wardrobe. It's easy to accumulate clothes and completely forget what you own — or worse, keep buying duplicates of things you already have. I wanted a simple, visual, and fun way to manage my outfits digitally, help myself get dressed with less stress, and make the most of the clothes I already love.

It was also a wonderful opportunity to build something personal and meaningful while practicing modern web development. This project is close to my heart — it solves a real, everyday problem in the cutest way I could manage!

---

## Tech Stack

This app is built with a modern, lightweight stack:

- **React** — for a smooth, interactive UI
- **TypeScript** — for type safety and confidence
- **Vite** — for a blazing fast development experience
- **Tailwind CSS** — for beautiful, responsive styling
- **shadcn/ui** — for polished, accessible UI components

## Run it locally

```sh
# Clone the repo
git clone https://github.com/richelleadarlo/digital-closet.git

# Navigate into the project
cd digital-studio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open your browser and head to `http://localhost:8080` 

---

## Running Tests

```sh
npm run test
```
